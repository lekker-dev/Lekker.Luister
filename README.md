# Lekker.Luister
A "Lekker" web app to get some music information from music streaming services such as Deezer, Spotify etc. through their open API's and implementing it with Angular and .net Core.

Lekker (Afrikaans) - nice, sweer, good, enjoyable, dope... the list goes on...

Luister (Afrikaans) - to listen.
